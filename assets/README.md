# Assets

Static media — images, diagrams, and video used in documentation or shared to show the project off.

| Folder | Purpose |
|---|---|
| `images/` | Build photos, screenshots, figures |
| `diagrams/` | Architecture and dataflow diagrams |
| `media/` | Demo videos and recordings |

Not for sensor data or datasets — those are [data/](../data/).

## Commit diagram sources

Export the PNG *and* commit the editable source next to it (`.drawio`, `.excalidraw`, or the Mermaid text). A diagram you can't edit is a diagram that goes stale and then goes wrong, which is worse than not having one.

## Photograph the build as you go

Take photos while wiring, not after. In eighteen months, "which way round did that connector go?" is a question a five-second photo answers and your memory doesn't.
